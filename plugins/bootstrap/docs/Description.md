# Bootstrap 5 Components Plugin

## Overview

The Bootstrap plugin integrates the Bootstrap 5 CSS framework into MD++, enabling you to use Bootstrap UI components directly in your Markdown documents. This includes cards, alerts, badges, buttons, modals, tabs, grid layouts, and many more components.

## Features

- **40+ Components** - Full Bootstrap 5 component library
- **Grid System** - Responsive 12-column grid layout
- **Variants** - Color and size variants for most components
- **Nesting** - Components can be nested inside each other
- **Responsive** - Mobile-first responsive design
- **Interactive** - JavaScript-powered components (modals, tabs, dropdowns)
- **Accessibility** - ARIA-compliant components

## Component Categories

### Layout Components
- Container, Row, Col (Grid System)
- Flexbox utilities

### Content Components
- Cards (with header, body, footer)
- Tables (striped, bordered, hover)
- Alerts and Badges
- Progress bars

### Navigation Components
- Navbar with brand and links
- Tabs and Pills
- Breadcrumbs
- Dropdowns

### Interactive Components
- Modals (dialog boxes)
- Accordion (collapsible panels)
- Carousel (image slider)
- Offcanvas (sidebar)
- Toast notifications

### Form Components
- Form groups
- Form controls
- Form labels

## Assets

The plugin loads Bootstrap 5 from CDN:
- **CSS**: `bootstrap@5.3.0/dist/css/bootstrap.min.css`
- **JS**: `bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js`

## Requirements

- MD++ Editor version 1.0.0 or higher
- Internet connection (for loading Bootstrap from CDN)

## Color Variants

Most components support Bootstrap's color system:

| Variant | Color | Use Case |
|---------|-------|----------|
| `primary` | Blue | Primary actions |
| `secondary` | Gray | Secondary actions |
| `success` | Green | Positive states |
| `danger` | Red | Errors, destructive |
| `warning` | Yellow | Cautions |
| `info` | Cyan | Information |
| `light` | Light gray | Light backgrounds |
| `dark` | Dark gray | Dark backgrounds |
