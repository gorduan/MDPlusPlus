# Tailwind CSS Components Plugin

## Overview

The Tailwind plugin brings utility-first CSS styling to MD++ through pre-built components. Use directive syntax to create cards, alerts, badges, buttons, grids, and more with Tailwind CSS classes. Components are styled with Tailwind's modern design system and support dark mode.

## Features

- **Pre-built Components** - Ready-to-use UI components
- **Utility Classes** - Tailwind CSS utility-first approach
- **Dark Mode Support** - Automatic dark/light mode styles
- **Responsive Design** - Mobile-first responsive components
- **Variants** - Color and style variants for components
- **Flexible Layouts** - Grid and flexbox layout components

## Available Components

| Component | Description |
|-----------|-------------|
| `card` | Content card with header and body |
| `alert` | Notification message boxes |
| `badge` | Small labels and tags |
| `button` | Interactive buttons |
| `container` | Centered content wrapper |
| `grid` | Responsive grid layouts |
| `flex` | Flexbox containers |
| `divider` | Horizontal separator |
| `quote` | Styled blockquote |
| `code` | Code block container |
| `highlight` | Text highlighter |

## Color Variants

Most components support these variants:

| Variant | Color |
|---------|-------|
| `primary` | Blue |
| `secondary` | Gray |
| `success` | Green |
| `warning` | Yellow |
| `error` / `danger` | Red |
| `info` | Cyan |
| `default` | Neutral gray |

## Assets

The plugin loads Tailwind CSS from CDN:
- `https://cdn.tailwindcss.com`

Note: The CDN version is intended for development. For production, consider a custom Tailwind build.

## Requirements

- MD++ Editor version 1.0.0 or higher
- Internet connection (for loading Tailwind from CDN)
